Himno del SENA

Estudiantes del SENA adelante Por Colombia luchad con amor Con el animo noble y radiante Transformémosla en mundo mejor I De la patria el futuro destino, en las manos del joven está, el trabajo es seguro camino, que el progreso a Colombia dará. II En la forja del SENA se forman, hombres libres que anhelan triunfar, con la ciencia y la técnica unidas, nuevos rumbos de paz trazarán. III Hoy la patria nos grita sentida, ¡estudiantes del SENA triunfad! solo así lograréis en la vida, más justicia, mayor libertad. IV Avancemos con fuerza guerrera, ¡estudiantes con firme tesón! que la patria en nosotros espera, su pacífica revolución.

Glosario de términos HTML

HTML
Lenguaje de marcado para la elaboración de páginas web.
CSS
Hojas de estilo en cascada usadas para describir la presentación de un documento HTML.
JavaScript
Lenguaje de programación interpretado, dialecto del estándar ECMAScript.
DOM
Modelo de Objeto de Documento, una interfaz de programación para documentos HTML y XML.
API
Interfaz de Programación de Aplicaciones, un conjunto de subrutinas, funciones y procedimientos.
HTTP
Protocolo de Transferencia de Hipertexto, usado en cada transacción de la World Wide Web.
URL
Localizador de Recursos Uniforme, la dirección específica que se asigna a cada uno de los recursos disponibles en la red.
SEO
Optimización para motores de búsqueda, el proceso de mejorar la visibilidad de un sitio web en los resultados de los buscadores.
Responsive
Diseño web adaptable, que busca la correcta visualización de una misma página en distintos dispositivos.
Framework
Marco de trabajo, un conjunto estandarizado de conceptos, prácticas y criterios para enfocar un tipo de problemática particular.
